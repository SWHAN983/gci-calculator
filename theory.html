Grid Convergence Index
이론 및 계산 가이드

STAR-CCM+ Grid Sensitivity Test 적용 · Celik et al. (2008) ASME 표준 기반
CFD 수치 해석에서 격자 이산화 불확실도를 정량적으로 추정하는 방법론입니다.

01 · 개요

GCI란 무엇인가?

격자(Mesh) 크기로 인한 수치 오차를 보수적으로 정량화하는 표준 방법입니다.

핵심 질문
"현재 사용 중인 격자로 얻은 해가 격자 무관(grid-independent) 해에 얼마나 가까운가?"

CFD 해석에서 격자를 무한히 세밀하게 만들수록 수치 해는 이론적 정확해에 수렴합니다. GCI는 현실적인 계산 자원으로 사용하는 격자에서 발생하는 이산화 불확실도를 95% 신뢰 수준으로 추정합니다.

02 · 핵심 개념

이산화 오차와 수렴 차수

수치 해는 격자 크기 h가 작아질수록 정확해에 가까워집니다.

수치 해의 구조
\[ \phi_{\text{numerical}} = \phi_{\text{exact}} + C \cdot h^p + O(h^{p+1}) \]

\[ h_{3D} = \left(\frac{V_{\text{domain}}}{N}\right)^{1/3} \qquad h_{2D} = \left(\frac{A_{\text{domain}}}{N}\right)^{1/2} \]

03 · 계산 절차

5단계 GCI 계산 절차

세 격자(Coarse / Medium / Fine)의 결과로부터 순서대로 계산합니다. h₁ < h₂ < h₃

01
격자가 얼마나 촘촘한가?
격자 정제 비율
\[ r_{21} = \frac{h_2}{h_1}, \qquad r_{32} = \frac{h_3}{h_2} \]
권장 기준: r ≥ 1.3. 비율이 너무 작으면 격자 간 차이가 작아 수렴 차수를 정확히 계산할 수 없습니다.
02
얼마나 빠르게 수렴하는가?
관측 수렴 차수 p (고정점 반복법)
\[ p = \frac{1}{\ln r_{21}} \left| \ln\!\left|\frac{\varepsilon_{32}}{\varepsilon_{21}}\right| + \ln\!\left(\frac{r_{21}^p - s}{r_{32}^p - s}\right) \right| \]
\[ \varepsilon_{21} = \phi_2 - \phi_1, \quad \varepsilon_{32} = \phi_3 - \phi_2, \quad s = \operatorname{sgn}\!\left(\frac{\varepsilon_{32}}{\varepsilon_{21}}\right) \]
수렴 판정: ε₃₂/ε₂₁ > 0 → 단조 수렴(신뢰 가능)  |  ε₃₂/ε₂₁ < 0 → 진동 수렴(신뢰도 낮음)
STAR-CCM+ 2차 기법의 이론값: p ≈ 2
03
완벽한 격자라면 결과가 얼마일까?
Richardson 외삽 — 격자 독립 해 추정
\[ \phi_{\text{ext}}^{21} = \frac{r_{21}^p\,\phi_1 - \phi_2}{r_{21}^p - 1} \]
격자 크기를 0으로 보낸 경우의 이론적 해를 수학적으로 추정한 값입니다.
04
현재 격자 오차는 최대 몇 %?
GCI 계산 — 핵심 단계
\[ e_a^{21} = \left|\frac{\phi_1 - \phi_2}{\phi_1}\right| \times 100\% \]
\[ \mathrm{GCI}_{\text{fine}}^{21} = \frac{F_s \cdot e_a^{21}}{r_{21}^p - 1}, \qquad F_s = 1.25 \text{ (3개 격자 표준)} \]
안전계수 Fₛ = 1.25를 포함한 보수적 오차 추정값입니다. 95% 신뢰 수준으로 격자 이산화 불확실도를 정량화합니다.
05
이 GCI 값 자체를 믿을 수 있는가?
점근 수렴 범위 검증
\[ \text{Check} = \frac{\mathrm{GCI}_{32}}{r_{21}^p \cdot \mathrm{GCI}_{21}} \approx 1.0 \]
이 값이 1.0 ± 0.05 범위 내이면 세 격자 모두 수치 점근 범위에 진입해 있으며, GCI 계산 결과의 신뢰도가 높습니다.
04 · 결과 해석

GCI 값 판정 기준

GCI_fine = 2.5% → Fine 격자 결과가 격자 독립 해로부터 최대 2.5% 이내 (95% 신뢰 수준)

GCI_fine 값판정권장 조치
< 1%🟢 우수격자 독립성 달성. 결과를 신뢰할 수 있음
1 – 3%🟡 양호공학적으로 수용 가능. 불확실도를 보고서에 명시
3 – 5%🟠 보통추가 정제 권장. 국소 정제(Local refinement) 적용 고려
> 5%🔴 불량격자 독립성 미달. Fine 격자 대폭 정제 필요
05 · 주의사항

GCI의 한계와 적용 조건

① 오차 추정기이지 정확한 오차가 아닙니다. 이산화 오차만 추정하며, 난류 모델 오차 등 모델링 오차는 포함하지 않습니다.
② 점근 수렴 범위 진입이 전제됩니다. 격자가 아직 수치 점근 범위에 진입하지 않으면 GCI가 과대 또는 과소 추정될 수 있습니다.
③ 관측 차수 p 범위를 확인하세요. 일반적으로 0 < p ≤ p_formal + 1 범위를 벗어나면 점근 범위 미진입으로 판단합니다.
④ 진동 수렴 시 주의. ε₃₂/ε₂₁ < 0이면 GCI 신뢰성이 떨어집니다. 격자 전략을 재검토하세요.
